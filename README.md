**MetaScraper**

Retreives Title, Description and OpenGraph images and returns back the data so it can be embeded as a preview in your web application.


**Module Dependacny**

pip install flask

pip install pymongo

pip install asyncio

pip install aiohttp

pip install bs4

pip install base64

pip install praw

pip install tldextract

pip install lxml
